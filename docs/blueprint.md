# **App Name**: CLASH ARENA

## Core Features:

- Arena Tournament Engine: A categorized tournament registration system based on Town Hall levels with coin-based entry fees and Prize Pool management.
- Dynamic Bracket Visualization: Real-time responsive tournament brackets supporting 8, 16, and 32-player single-elimination paths.
- Automated Result Verification Tool: Generative AI vision tool to analyze Cloudinary-hosted result screenshots and confirm match scores vs reported data to assist admin verification.
- Coin Ledger & Wallet: Non-withdrawable coin system for managing entry fees and distribution of winnings (1 Coin = 1 INR).
- Hall of Champions Ledger: A transparency record featuring past winners, rewards distributed, and match histories using Firestore real-time updates.
- Advanced Admin Command Center: Secure dashboard to moderate users, adjust wallet balances, create brackets, and manage site-wide announcements.
- Esports Identity Profile: Customizable user profiles tracking Player Tags, wins, losses, and specific Town Hall accomplishments via Google Auth.

## Style Guidelines:

- Primary: Energetic Flame Orange (#FF4500) symbolizing battle intensity and action.
- Background: Deep Obsidian Charcoal (#0F0E0D) to emphasize esports prestige and visual depth.
- Accent: Crimson Combat Red (#DC143C) for destructive highlights and critical status indicators.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, high-tech esports feel. Body font: 'Inter' (sans-serif) for legibility in dense tournament tables.
- Mobile-first floating bottom navigation with glassmorphism effects, accompanied by a persistent high-utility fixed header.
- Bold, gaming-inspired vector icons for 'Arena', 'Trophies', and 'Shields' with subtle outer glows.
- Liquid-smooth CSS transitions for the slide-out menu and floating ambient particle backgrounds to create a 'living' digital arena.
